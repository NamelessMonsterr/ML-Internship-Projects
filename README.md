
# Machine Learning Internship Projects

This repository contains machine learning and deep learning tasks completed during my internship. The projects cover different areas of ML including regression, clustering, classical machine learning, and deep learning for image classification.

## Projects Included

### 1. House Price Prediction
- Dataset: Kaggle House Prices
- Model: Ridge Regression
- Objective: Predict house sale prices using structured data features.

### 2. Customer Segmentation
- Algorithm: K-Means Clustering
- Objective: Group customers based on behavioral patterns.

### 3. Dogs vs Cats Classification
- Model: Support Vector Machine (SVM)
- Objective: Classify images as cat or dog using image features.

### 4. Hand Gesture Recognition
- Model: Convolutional Neural Network (CNN)
- Dataset: LeapGestRecog
- Objective: Recognize hand gestures from image inputs.

### 5. Food Image Classification
- Model: MobileNetV2 (Transfer Learning)
- Dataset: Food-101
- Objective: Classify food images into multiple categories.
